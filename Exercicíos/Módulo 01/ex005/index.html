Como usar o negrito e o itálico do jeito certo.

Texto negrito

Neste texto tem um termo em negrito usando a tag STRONG (semântica).

Texto itálico

Neste texto tem um termo em itálico usando a tag EM (semântica).

As tags I e B NÃO são semânticas no HTML5.

Técnica do Guanabara

Técnica do Guanabara: CTRL + SHIFT + P = abre o buscador de comandos, depois procurar abb e depois a função que quer colocar na parte desejada. (Antes precisa marcar a parte do texto que vc quer colocar a função.).

Texto marcado

Para deixar um texto marcado basta usar a tag mark. Para mudar a cor do marcador coloca a tag style dentro dentro da tag mark ou cria um seletor na área head.

Texto deletado

Para riscar um texto basta colocar a tag del.

Texto inserido

Para sublinhar um texto basta colocar a tag ins.

Texto sobrescrito

Para sobrescrever um texto/número basta usar a tag. Ex.: sup X20+3

Texto subscrito

Para subscrever um texto basta usar a tag sub. Ex.: H2O
